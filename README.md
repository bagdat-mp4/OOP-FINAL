# KBTU University Management System

OOP финальный проект. Система управления университетом, есть консоль и GUI на JavaFX.

## Как запустить

```bash
mvn clean javafx:run
```

## Что умеет каждая роль
**Admin** — создаёт/удаляет пользователей, меняет пароли, смотрит логи

**Teacher** — ставит оценки (AT1 макс 30, AT2 макс 30, Final макс 40), пишет жалобы менеджеру, рекомендательные письма, отмечает посещаемость

**Student** — записывается на курсы, смотрит транскрипт и GPA, оценивает преподавателей, вступает в организации

**Graduate Student** — всё как у студента + назначить научрука (его H-index должен быть >= 3) + добавлять дипломные проекты

**Manager** — добавляет курсы, назначает преподавателей, рассылает сообщения всем сотрудникам, видит жалобы, назначает researcher'ов

**Tech Support** — принимает/отклоняет заявки, при изменении статуса сотрудник получает уведомление в inbox

**Researcher** — это не отдельная роль, любого пользователя менеджер может сделать researcher'ом. тогда появляется раздел с статьями, проектами, H-index, citation форматы
